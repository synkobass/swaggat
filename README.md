
# Sagat Skill/Trigger Matchup Guide

## Introduction

This guide is to help new Sagat players to pick the right V-Skill and V-Trigger for the specific matchups. 

General matchups are devided into 4 tiers:
* advantageous
* even
* disadvantageous
* very disadvantageous
